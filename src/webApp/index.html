项目主页：github

已实现的接口

  • /lostInfo
  • /findInfo

    GET，获取拾取物品信息

    request:{ }

    response:{ "data":map的列表, "mas":"SUCCESS/系统开小差了" }

  • /login

    POST，登陆

    request:{ "name":"xiaoxiao", "password":"88888888" }

    response:{ "msg":"msg":"SUCCESS/FAILD/WRONG PASSWORD/NO SUCH USER", "token":"一个25位的字符串" }

  • /img?action=?
  • /searchInfo
